# flutter_agora
flutter播客应用。

## 自动化打包上传pgyer

批处理脚本[upload_pgyer.sh](upload_pgyer.sh)，包装Flutter build命令，并上传蒲公英。
后续通过执行该sh文件即可。
```shell
./upload_pgyer.sh
```
